# Idea Refine

Phase 1 (Expand): Restate as HMW, ask questions, generate 5-8 variations.
Phase 2 (Converge): Cluster into 2-3 directions, stress-test, surface assumptions.
Phase 3 (Ship): One-pager: Problem, Direction, Assumptions, MVP, Not Doing.
Save to docs/ideas/[name].md.
