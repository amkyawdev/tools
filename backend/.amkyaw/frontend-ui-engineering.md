# Frontend UI Engineering

Production-quality UIs: accessible, performant, visually polished.

State: useState → lifted → Context → URL → React Query → global.
Avoid AI Aesthetic: no purple gradients, excessive rounding, stock grids.
Accessibility (WCAG 2.1 AA): keyboard nav, ARIA, focus, empty/error states.
Responsive: mobile-first, test 320/768/1024/1440. Loading: skeleton, not spinner.
